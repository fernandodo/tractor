# Vim 移动命令大全（最系统的分类）

本文件整理了 Vim 中最完整、结构化、易查阅的移动命令集合，按类别分组，适合作为独立速查表使用。

---

# Ⅰ. 基础移动（Basic Motions）

### 行内移动
- [x] `h`：左  
- [x] `l`：右  
- [x] `j`：下  
- [x] `k`：上  
- [x] `0`：行首  
- [ ] `^`：行首第一个非空字符  
- [x] `$`：行尾  

---

# Ⅱ. 单词移动（Word Motions）

### 基于“单词”（word）
- [x] `w`：下一个单词开头  
- [x] `e`：当前/下个单词结尾  
- [x] `b`：上一个单词开头  
- [ ] `ge`：上一个单词结尾  

### 基于“WORD”（含符号整体）
- [ ] `W`：下一个 WORD 开头  
- [ ] `E`：下一个 WORD 结尾  
- [ ] `B`：上一个 WORD 开头  
- [ ] `gE`：上一个 WORD 结尾  

---

# Ⅲ. 行内字符查找（f/t 系列）

- [ ] `fx`：跳到下一个 `x`  
- [ ] `Fx`：跳到上一个 `x`  
- [ ] `tx`：跳到下一个 `x` 前一个字符  
- [ ] `Tx`：跳到上一个 `x` 后一个字符  
- [ ] `;`：重复上次 f/F/t/T  
- [ ] `,`：反向重复  

---

# Ⅳ. 跳转（Jump Motions）

### 括号/结构跳转
- [ ] `%`：匹配跳转括号、if-else 等结构  

### 定义跳转
- [x] `gd`：跳到局部定义（VSCodeVim 映射为 LSP）  
- [x] `gD`：跳到全局定义  

### 位置历史跳转
- `Ctrl-o`：向后跳转（返回）  
- `Ctrl-i`：向前跳转  
- [ ] \`\`：返回上一次光标的精确位置（极常用）  

---

# Ⅴ. 行号移动

- [x] `gg`：跳到文件开头  
- [ ] `G`：跳到文件末尾  
- [ ] `nG`：跳到第 n 行（如 `50G`）  
- [ ] `:n`：跳到行号 n  

---

# Ⅵ. 翻页（Scrolling）

### 整页
- `Ctrl-f`：下一页（向下翻）  
- `Ctrl-b`：上一页（向上翻）  

### 半页
- `Ctrl-d`：下半页  
- `Ctrl-u`：上半页  

### 单行滚动（光标不动）
- `Ctrl-e`：向下滚  
- `Ctrl-y`：向上滚  

### 屏幕定位
- [ ] `zz`：当前行置中  
- [ ] `zt`：当前行置顶  
- [ ] `zb`：当前行置底  

---

# Ⅶ. 搜索移动（Search Motions）

- [ ] `/text`：向下搜索  
- [ ] `?text`：向上搜索  
- [ ] `n`：下一个匹配  
- [ ] `N`：上一个匹配  
- [ ] `*`：搜索光标所在词（向下）  
- [ ] `#`：搜索光标所在词（向上）  

---

# Ⅷ. 大范围移动（段落/句子/结构）

### 按段落（由空行分隔）
- [ ] `{`：上一个段落  
- [ ] `}`：下一个段落  

### 按句子（以 `.?!` 分隔）
- [ ] `(`：上一句  
- [ ] `)`：下一句  

### 按代码块（大括号）
- [ ] `[[`：上一个 `{` 或函数开头  
- [ ] `]]`：下一个 `{` 或函数开头  
- [ ] `[]`：上一个 `}`  
- [ ] `][`：下一个 `}`  

---

# Ⅸ. 标记（Marks）

- [ ] `ma`：设置标记 a  
- [ ] `'a`：跳到标记 a 所在行  
- [ ] ``a`：跳到标记 a 的精确位置  
- [ ] ``.``：跳回上次修改位置  
- [ ] ``"``：回到上次退出文件的位置  

---

# Ⅹ. 屏幕坐标移动（Screen Motions）

- [ ] `H`：跳到屏幕顶部  
- [ ] `M`：跳到屏幕中间  
- [ ] `L`：跳到屏幕底部  

---

# 汇总结构（可打印版）

```
基本： h j k l 0 ^ $
单词： w e b ge (W E B gE)
查找： f t F T ; ,
跳转： % gd gD Ctrl-o Ctrl-i ``
行号： gg G nG :n
翻页： Ctrl-f Ctrl-b Ctrl-d Ctrl-u Ctrl-e Ctrl-y zz zt zb
搜索： / ? n N * #
段落： { }
句子： ( )
结构： [[ ]] [] ][
标记： ma 'a `a `. `"
屏幕： H M L
```
